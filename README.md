### **[Reconciler CLI](https://github.com/anibalxyz/reconciler)** (Part of Reconciler)

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![Typer](https://img.shields.io/badge/Typer-000000?style=flat-square)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat-square&logo=docker&logoColor=white)
![Docker Compose](https://img.shields.io/badge/Docker_Compose-2496ED?style=flat-square&logo=docker&logoColor=white)

**Description:** Command-line interface for managing Reconciler's Docker environment and automating development workflows. Built with Python and Typer, following a modular architecture for easy extension.

Features include:

- **Simplified Docker management:** Abstracts complex `docker compose` commands with multiple flags and environment configurations
- **Environment switching:** Seamlessly switch between dev, test, and prod environments
- **Automated testing:** Quick test execution with proper environment setup
- **Image management:** Build and tag Docker images with consistent naming
- **Modular design:** Clean separation of concerns (compose, image, resource, config modules) for maintainability
